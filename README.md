# Comdash

Comdash is a simple side-scroller built with [Processing](https://processing.org/) intended to be a simple example project for teaching git and programming basics.

## Build Instructions

1. Install [Processing](https://processing.org/) (version 4.5)
2. Run Processing at least once (it just needs to start, you can directly close the Processing IDE again)
3. Install [Visual Studio Code](https://code.visualstudio.com/)
4. Install the official [Processing extension](https://marketplace.visualstudio.com/items?itemName=processing-foundation.processing-vscode-extension)
5. Open this folder in VS Code
6. Run the sketch via the extension (you should see a play button in the top right corner if you have any `.pde` file open)

## Teaching Remarks

- For teaching purposes this code is always using `this.` to access member variables or call member functions to make it clear where they come from, even though it is not required
- Normally you should put source files into a subdirectory instead of the repository root but Processing is a bit special
